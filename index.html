Car Flip Tracker

A full-stack PERN app built for my dad who flips used cars — buying damaged vehicles at IAA auctions, repairing them, and reselling. Includes a live bid calculator, VIN decoder, AI-powered damage estimation via the Claude API, and a church donation tracker (he donates 35% of every profit).

React Node.js Express PostgreSQL Cloudinary Claude API NHTSA API Railway Vercel
Live Demo ↗ GitHub ↗
VIN decoder — paste a VIN, auto-fills make, model, year & drivetrain via NHTSA API
AI damage estimator — uploads photos to Claude API, returns a parts list with cost estimates
Live bid calculator using the 95% KBB trade-in formula to find the max safe auction bid
Parts tracker with per-car running totals that update financials in real time
Status workflow: evaluating → active → sold → passed
Dashboard showing total profit, flip history, and church donation totals
Deployed on Railway (backend + DB) and Vercel (frontend)
